# Architettura

Ora che hai completato l'Inizio Rapido e hai in mano un'applicazione Tauri base, ti senti tentato di buttarti subito in fondo. Ti invitiamo, tuttavia, a resistere a questa tentazione per un paio di pagine in più e ad imparare di più sui concetti e le idee alla base di Tauri. Troverai che lo sviluppo di un app è molto simile alla creazione di una applicazione client-server tradizionale sul web, con un paio di piccole ma non tantomento importanti differenze.

This section will cover Tauri's multi-process architecture, windows, and webviews and our design decisions to make your application more secure and resource-efficient.
