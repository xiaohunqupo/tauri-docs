# Guide

Tauri è un toolkit di costruzione app che consente di costruire software per tutti i principali OS desktop utilizzando le tecnologie web.

Che tu sia nuovo allo sviluppo di Tauri, voglia integrare Tauri in un progetto esistente, sia pronto a mettere la tua app nelle mani degli utenti, o tu voglia semplicemente imparare come usare una caratteristica comune di Tauri, abbiamo aiuto per te.

Se non riesci a trovare quello che stai cercando o hai un suggerimento per migliorare le nostre guide, raggiungi la nostra comunità su [Discord](https://discord.com/invite/tauri).
