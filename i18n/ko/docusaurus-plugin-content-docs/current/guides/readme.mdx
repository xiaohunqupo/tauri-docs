# 안내서

Tauri는 웹 기술을 사용해 주요 데스크톱 운영체제용 소프트웨어를 만들게 돕는 앱 개발 툴킷입니다.

Tauri 개발이 처음이시거나, 기존 프로젝트에 Tauri를 연동하고 싶으시거나, 사용자에게 앱을 제공할 준비가 되었거나, Tauri 기능의 사용법을 알고 싶으실 때, 그런 당신을 위한 도움말이 여기에 있습니다.

찾으시는 자료가 없거나 안내서를 개선할 제안 사항이 있으시다면 [Discord](https://discord.com/invite/tauri) 커뮤니티로 연락주세요.
