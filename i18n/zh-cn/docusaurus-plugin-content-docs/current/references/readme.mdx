# 架构

现在，您已完成了快速上手的步骤并开发了一款简单的 Tauri 应用，现在是时候深入探索了！ 但是，在进一步学习 Tauri 之前，我们恳切地想请您先抑制住内心的冲动，先阅读和了解一些更多关于它背后所蕴含的理念和思考。 您会发现，开发一款 Tauri 应用和开发一个传统的客户端 - 服务器应用相同，只是多了几个微妙又重要的不同。

此章节会概述 Tauri 的多进程架构、窗口、WebView 以及让您的应用既安全又高效利用资源的设计和决策。
