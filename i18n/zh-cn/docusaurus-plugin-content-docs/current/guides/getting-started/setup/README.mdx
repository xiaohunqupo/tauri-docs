# 快速开始

Tauri 兼容**绝大多数前端工具栈**。 选择您最喜欢的框架，然后开始吧！

:::tip `create-tauri-app`

我们推荐您使用 [`create-tauri-app`][] 来轻松快速地创建一个新 Tauri 项目。 它为原生 HTML/CSS/JavaScript 及如 React、Svelte 和 Yew 一类的框架提供了脚手架模板。

<CreateTauriApp />

注意，若您使用 `create-tauri-app` 的话则无需遵循下列教程中的指引。但我们仍建议您阅读 (如 [HTML/CSS/JavaScript][] 指南) 来理解配置流程。

:::

若您不熟悉网页开发，或没有偏好的前端框架，您可参阅 [HTML/CSS/JavaScript][] 指南。 该教程指导您使用 Node 或 Cargo 进行最小化的前端环境配置。
